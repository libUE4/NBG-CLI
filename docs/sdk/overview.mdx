NBG SDK 是用于构建 agentic applications 的开源框架。当前 SDK 包名仍沿用 `@cline/*` 兼容层，但运行时能力与 NBG CLI 和 IDE 集成共享同一套底座。它使用 plugin architecture，支持 checkpoints、web fetch、MCP、cron jobs、subagents 等 agent 常用能力。

你可以使用 NBG SDK 在 CI/CD pipelines 中运行 agents，创建端到端 workflow automations，或把 agents 直接嵌入自己的产品。

## 安装

```bash
npm install @cline/sdk
```

`@cline/sdk` 会导出所有 SDK packages：`@cline/core` 提供完整 agent harness，`@cline/agents` 提供 stateless agent loop，`@cline/llms` 控制 model gateway，`@cline/shared` 提供通用 utilities。

需要 Node.js 22 或更高版本。

## SDK Skill

如果你使用 coding agent（Claude Code、Codex 等），可以安装当前兼容的 [SDK skill](https://github.com/cline/sdk-skill)，让 agent 获取 SDK API 和最佳实践上下文。该 skill 包名目前仍沿用兼容名称。

```bash
npx skills add cline/sdk-skill
```

可以让它帮你 scaffold agents、创建 custom tools、接入 plugins、配置 providers 等。

## 第一个 Agent

```typescript
import { Agent } from "@cline/sdk"

const agent = new Agent({
  providerId: "anthropic",
  modelId: "claude-sonnet-4-6",
  apiKey: process.env.ANTHROPIC_API_KEY,
  maxIterations: 1,
})

agent.subscribe((event) => {
  if (event.type === "assistant-text-delta") {
    process.stdout.write(event.text ?? "")
  }
})

const result = await agent.run("Explain what an SDK is in two sentences.")
```

<Note>
  完整示例见 [quickstart example](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples/quickstart)。Clone 后运行 `bun dev` 即可试用。
</Note>

## Packages

| Package | 用途 |
|---------|---------|
| `@cline/sdk` | Public SDK surface（re-export `@cline/core`） |
| `@cline/core` | 面向 sessions、built-in tools、persistence、hub support、automation 的 Node runtime |
| `@cline/agents` | Browser-compatible stateless agent execution loop |
| `@cline/llms` | Provider gateway 和 model catalogs |
| `@cline/shared` | Types、schemas、tool helpers、hooks、storage helpers |



Package boundaries 和 exports 见 [Packages](/sdk/architecture/overview)。
