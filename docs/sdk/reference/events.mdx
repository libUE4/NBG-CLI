SDK 暴露多个 event surfaces。

## AgentRuntimeEvent

直接使用 `AgentRuntime` 时，会通过 `agent.subscribe(listener)` 发出 low-level runtime events。

```typescript
const unsubscribe = agent.subscribe((event) => {
  console.log(event.type)
})
```

常见 runtime events 包括 model events、tool execution events、run lifecycle events 和 failure events。Runtime results 使用 `AgentRunResult.status`，而不是 `finishReason`。

## AgentEvent

Core/host-facing agent events 通过 `AgentConfig.onEvent` 和 core adapters 发出。

| Event | 说明 |
|-------|-------------|
| `content_start` | Text/reasoning/tool content 开始 |
| `content_update` | Tool progress update |
| `content_end` | Text/reasoning/tool content 完成 |
| `iteration_start` | Loop iteration 开始 |
| `iteration_end` | Loop iteration 完成 |
| `usage` | Token/cost usage update |
| `notice` | Runtime status/recovery notice |
| `done` | Agent completed/aborted/failed |
| `error` | Error occurred |

## Done Event

```typescript
interface AgentDoneEvent {
  type: "done"
  reason: "completed" | "max_iterations" | "aborted" | "mistake_limit" | "error"
  text: string
  iterations: number
  usage?: LegacyAgentUsage
}
```

## Usage Event

```typescript
interface AgentUsageEvent {
  type: "usage"
  inputTokens: number
  outputTokens: number
  cacheReadTokens?: number
  cacheWriteTokens?: number
  cost?: number
  totalInputTokens: number
  totalOutputTokens: number
  totalCacheReadTokens?: number
  totalCacheWriteTokens?: number
  totalCost?: number
}
```

## CoreSessionEvent

`ClineCore.subscribe(listener, options?)` 会从 runtime host 发出 session-level events。

```typescript
const unsubscribe = cline.subscribe((event) => {
  console.log(event.type, event.sessionId)
}, { sessionId })
```
