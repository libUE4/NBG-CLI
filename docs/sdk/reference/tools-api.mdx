## `createTool(config)`

```typescript
import { createTool } from "@cline/sdk"
```

创建 typed tool。它也从 `@cline/agents` 和 `@cline/core` re-export。

```typescript
const tool = createTool({
  name: "get_current_time",
  description: "Return the current time as ISO string.",
  inputSchema: { type: "object", properties: {} },
  execute: async (_input, context) => {
    return { now: new Date().toISOString() }
  },
})
```

`inputSchema` 可以是 JSON Schema，也可以是 Zod schema。

## AgentTool

```typescript
interface AgentTool<TInput = unknown, TOutput = unknown> {
  name: string
  description: string
  inputSchema: Record<string, unknown>
  execute: (input: TInput, context: AgentToolContext, onChange?: (update: unknown) => void) => Promise<TOutput>
  timeoutMs?: number
  retryable?: boolean
  maxRetries?: number
}
```

`createTool` 的默认值：

| Field | Default |
|-------|---------|
| `timeoutMs` | `30000` |
| `retryable` | `true` |
| `maxRetries` | `3` |

## AgentToolContext

```typescript
interface AgentToolContext {
  agentId: string
  conversationId: string
  iteration: number
  abortSignal?: AbortSignal
  metadata?: Record<string, unknown>
}
```

## ToolPolicy

```typescript
interface ToolPolicy {
  enabled?: boolean
  autoApprove?: boolean
}
```

Policy map 中没有列出的 tool names 默认 enabled 且 auto-approved。

## ToolCallRecord

```typescript
interface ToolCallRecord {
  id: string
  name: string
  input: unknown
  output: unknown
  error?: string
  durationMs: number
  startedAt: Date
  endedAt: Date
}
```
