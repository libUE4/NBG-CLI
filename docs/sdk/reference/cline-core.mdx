## `ClineCore.create(options)`

```typescript
const cline = await ClineCore.create({
  clientName: "my-app",
  backendMode: "auto",
})
```

常用 options：

| Option | Type | 说明 |
|--------|------|-------------|
| `clientName` | `string` | Human-readable SDK client name |
| `distinctId` | `string` | 稳定 telemetry/user identifier |
| `backendMode` | `"auto" \| "local" \| "hub" \| "remote"` | Runtime backend selection |
| `hub` | `HubOptions` | Local hub connection options |
| `remote` | `RemoteOptions` | Remote hub options |
| `capabilities` | `RuntimeCapabilities` | Client-owned tool executors 和 approval callbacks |
| `toolPolicies` | `Record<string, ToolPolicy>` | 默认 tool approval policies |
| `automation` | `boolean \| ClineCoreAutomationOptions` | 启用 automation APIs |
| `fetch` | `typeof fetch` | 用于 local provider calls 的 custom fetch |

## `start(input)`

```typescript
const session = await cline.start({
  prompt: "Summarize this repo",
  config: {
    providerId: "anthropic",
    modelId: "claude-sonnet-4-6",
    apiKey: process.env.ANTHROPIC_API_KEY,
    systemPrompt: "You are a helpful coding assistant.",
    cwd: process.cwd(),
    workspaceRoot: process.cwd(),
    enableTools: true,
    enableSpawnAgent: false,
    enableAgentTeams: false,
  },
})
```

### ClineCoreStartInput

| Field | Type | Required | 说明 |
|-------|------|----------|-------------|
| `prompt` | `string` | No | Initial prompt |
| `config` | `CoreSessionConfig` | Yes | Session model/runtime config |
| `source` | `SessionSource` | No | Session source label |
| `interactive` | `boolean` | No | Session 是否期待交互 |
| `sessionMetadata` | `Record<string, unknown>` | No | 随 session 持久化的 metadata |
| `initialMessages` | `Message[]` | No | 预加载 messages |
| `toolPolicies` | `Record<string, ToolPolicy>` | No | Per-session tool policies |
| `capabilities` | `RuntimeCapabilities` | No | Per-session tool executors 和 approval callbacks |

### StartSessionResult

```typescript
interface StartSessionResult {
  sessionId: string

```

## `send(input)`

```typescript
const result = await cline.send({
  sessionId,
  prompt: "Continue with tests",
})
```

返回 `AgentResult | undefined`。

## Session APIs

| Method | 说明 |
|--------|-------------|
| `subscribe(listener, options?)` | 订阅 `CoreSessionEvent` events |
| `list(limit?, options?)` | 列出 session history records |
| `listHistory(options?)` | 列出 session history records |
| `get(sessionId)` | 获取 session metadata |
| `readMessages(sessionId)` | 读取持久化 messages |
| `getAccumulatedUsage(sessionId)` | 读取累计 token/cost usage |
| `update(sessionId, updates)` | 更新 prompt/title/metadata |
| `abort(sessionId, reason?)` | 中止 active work |
| `stop(sessionId)` | 停止 session |
| `delete(sessionId)` | 删除 session |
| `restore(input)` | 从 checkpoint 恢复 |
| `dispose(reason?)` | 释放 runtime resources |

## Automation API

启用 `automation` 后，可以使用 `cline.automation` 启停 automation services、reconcile specs、ingest events，并列出 events/specs/runs。
