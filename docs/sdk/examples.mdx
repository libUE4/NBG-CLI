可运行示例位于 [SDK repository](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples)，并按难度组织：

| Example | 难度 | 说明 |
|---------|------------|-------------|
| [quickstart](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples/quickstart) | Beginner | 发送一个 prompt，并流式输出响应（约 15 行） |
| [cli-agent](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples/cli-agent) | Beginner | 带 shell tool 的交互式 terminal chat |
| [code-review-bot](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples/code-review-bot) | Intermediate | 带 custom tools 和 structured output 的 AI code reviewer |
| [multi-agent](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples/multi-agent) | Advanced | 使用 streaming web UI 并行运行多个 agents |
| [desktop-app](https://github.com/libUE4/NBG-CLI/tree/main/sdk/apps/examples/desktop-app) | Advanced | 完整 Tauri + Next.js desktop app |
