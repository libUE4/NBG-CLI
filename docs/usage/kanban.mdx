## 前置条件

- Node.js 18+
- Git 仓库（从 repo root 运行）
- Kanban launcher 可用（`npx kanban`）

## 快速开始

```bash
cd /path/to/your/repo
npx kanban
```

该命令会启动本地 server，并在浏览器中打开看板。

## 核心流程

1. **创建任务**：在看板上手动创建，或通过 sidebar chat 创建。
2. **启动任务**：点击 play 后，每张卡片都会获得隔离 git worktree 和 terminal。
3. **监控进度**：通过卡片状态和最新 agent output 跟踪。
4. **审查 diff**：在卡片详情中查看 diff，并留下 inline comments。
5. **交付变更**：通过 Commit 或 Open PR 完成。
6. **清理资源**：把卡片移动到 trash 后，对应 worktree 会被移除。

## 核心能力

- **并行执行：** 每个任务在独立 worktree 中运行，避免互相冲突。
- **任务关联：** dependency chains 可以自动启动后续任务。
- **Inline review loop：** 直接在 diff lines 上评论，引导 agent 修改。
- **Auto-commit / Auto-PR：** 可在 settings 中启用自动化。
- **Sidebar chat orchestration：** 让 agent 把工作拆成 cards 并启动流程。

## Agent 兼容性

Kanban 面向 CLI-style coding agents 设计。NBG Kanban 当前可与 NBG CLI、Claude Code、Codex、OpenCode，以及 settings 中可用的相关 runtimes 配合使用。
