## 什么是 TUI？

TUI 是 NBG 的交互式终端界面，适合在终端中对话式工作：提问、审查计划、批准操作并快速迭代。

## 前置条件

安装 CLI：

```bash
npm install -g nbg
```

## 启动 TUI

```bash
# 默认交互启动
nbg

# 显式 TUI 模式
nbg -i
```

## 核心 TUI 操作

- `Tab` 切换 Plan/Act
- `Shift+Tab` 切换全部自动批准
- `Ctrl+C` 中止正在运行的 turn；再次按下退出
- `Ctrl+D` 在 prompt 为空且空闲时退出
- `Ctrl+L` 清空 chat view 或当前 conversation
- `/settings`, `/model`, `/account`, `/mcp`, `/compact`, `/undo`, `/clear`, `/history`, `/help`, `/quit`
- 使用 `@file` mentions 引入 workspace context

状态区域会显示当前模型、上下文使用量、成本、workspace/branch、git diff 统计、Plan/Act 状态，以及是否启用全部自动批准。

## 何时使用 TUI 或 headless

- 需要协作、审查和批准时使用 **TUI**。
- 自动化场景使用 **headless**（`--json`、piped stdin/stdout、CI scripts）。

参考：[CLI headless mode](/usage/cli-overview#headless-mode)
