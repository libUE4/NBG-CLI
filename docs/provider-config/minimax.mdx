MiniMax 提供大上下文窗口和有竞争力的价格，代表模型包括 MiniMax-M2 系列。

**Website:** [https://www.minimax.io/](https://www.minimax.io/)

### 获取 API 凭证

1.  **注册或登录：** 打开 [MiniMax Platform](https://www.minimax.io/platform)，创建账号或登录现有账号。
2.  **进入 API Keys：** 找到 API key 管理区域。
3.  **创建密钥：** 生成新的 API key。
4.  **复制密钥：** 立即复制 API key，并妥善保存。

### 在 NBG 中配置

> 参考 [认证与模型选择](/getting-started/authorizing-with-cline#menu)。

1.  **打开 NBG 设置：** 在 NBG 面板中点击设置图标 (⚙️)。
2.  **选择 provider：** 在 "API Provider" 下拉菜单中选择 "MiniMax"。
3.  **填写 API key：** 粘贴 MiniMax API key。
4.  **选择模型：** 在 "Model" 下拉菜单中选择需要使用的模型。

### 提示

-   **大上下文：** 全部模型支持 192K token 上下文窗口。
-   **推理支持：** M2.7 和 M2.5 支持用于复杂任务的 extended thinking/reasoning。
-   **价格：** 当前费率以 [MiniMax pricing page](https://www.minimax.io/platform/document/pricing) 为准。
