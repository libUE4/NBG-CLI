Google Gemini 是 Google 的多模态 AI 模型系列，提供大上下文窗口，并在编码、推理和文档分析任务中表现稳定。

**Website:** [https://ai.google.dev/](https://ai.google.dev/)

### 获取 API 凭证

1.  **登录：** 打开 [Google AI Studio](https://aistudio.google.com/)，使用 Google 账号登录。
2.  **获取 API key：** 打开 [aistudio.google.com/apikey](https://aistudio.google.com/apikey)。
3.  **创建密钥：** 点击 "Create API Key"，选择或创建 Google Cloud project。
4.  **复制密钥：** 立即复制 API key，并妥善保存。

### 在 NBG 中配置

> 参考 [认证与模型选择](/getting-started/authorizing-with-cline#menu)。

1.  **打开 NBG 设置：** 在 NBG 面板中点击设置图标 (⚙️)。
2.  **选择 provider：** 在 "API Provider" 下拉菜单中选择 "Google Gemini"。
3.  **填写 API key：** 将 Google AI API key 粘贴到 "Gemini API Key" 字段。
4.  **选择模型：** 在 "Model" 下拉菜单中选择需要使用的模型。

### 提示

-   **价格：** 最新价格以 [Google AI pricing page](https://ai.google.dev/gemini-api/docs/models/gemini) 为准。
