**Website:** [https://platform.deepseek.com/](https://platform.deepseek.com/)

### 获取 API 凭证

1.  **注册或登录：** 打开 [DeepSeek Platform](https://platform.deepseek.com/)，创建账号或登录现有账号。
2.  **进入 API Keys：** 在平台的 [API keys](https://platform.deepseek.com/api_keys) 页面管理密钥。
3.  **创建密钥：** 点击 "Create new API key"，使用易识别的名称，例如 "NBG"。
4.  **复制密钥：** **重要：** 立即复制 API key。离开页面后通常无法再次查看，请妥善保存。

### 在 NBG 中配置

> 参考 [认证与模型选择](/getting-started/authorizing-with-cline#menu)。

1.  **打开 NBG 设置：** 在 NBG 面板中点击 ⚙️ 图标。
2.  **选择 provider：** 在 "API Provider" 下拉菜单中选择 "DeepSeek"。
3.  **填写 API key：** 将 DeepSeek API key 粘贴到 "DeepSeek API Key" 字段。
4.  **选择模型：** 在 "Model" 下拉菜单中选择需要使用的模型。
