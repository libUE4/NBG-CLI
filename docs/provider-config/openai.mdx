NBG 支持通过官方 OpenAI API 直接访问模型。

## 选择 OpenAI 接入方式

- **[OpenAI API (key-based)](#get-api-credentials)：** 使用 OpenAI API key。
- **[OpenAI Codex (Subscription OAuth)](#openai-codex-oauth)：** 使用 OpenAI 账号登录，通过订阅访问，无需管理 API key。

**Website:** [https://openai.com/](https://openai.com/)

## OpenAI API (key-based)

### 获取 API 凭证

1.  **注册或登录：** 打开 [OpenAI Platform](https://platform.openai.com/)，创建账号或登录现有账号。
2.  **进入 API Keys：** 登录后进入账号的 [API keys section](https://platform.openai.com/api-keys)。
3.  **创建密钥：** 点击 "Create new secret key"，建议使用易识别的名称，例如 "NBG API Key"。
4.  **复制密钥：** **重要：** 立即复制生成的 API key。出于安全原因，OpenAI 不会再次显示该密钥，请妥善保存。

### 在 NBG 中配置

> 参考 [认证与模型选择](/getting-started/authorizing-with-cline#menu)。

1.  **打开 NBG 设置：** 在 NBG 面板中点击设置图标 (⚙️)。
2.  **选择 provider：** 在 "API Provider" 下拉菜单中选择 "OpenAI"。
3.  **填写 API key：** 将 OpenAI API key 粘贴到 "OpenAI API Key" 字段。
4.  **选择模型：** 在 "Model" 下拉菜单中选择需要使用的模型。
5.  **可选，Base URL：** 如需通过代理或自定义 base URL 访问 OpenAI API，可在此处填写。大多数用户无需修改默认值。

### 提示

-   **价格：** 各模型价格请参考 [OpenAI Pricing page](https://openai.com/pricing)。
-   **Azure OpenAI Service：** 如需使用 Azure OpenAI，可参考对应文档，或在支持自定义 endpoint 的情况下按 OpenAI-compatible endpoint 配置。

## OpenAI Codex (OAuth)

1. 打开 NBG 设置。
2. 在 **API Provider** 中选择 **OpenAI Codex**。
3. 点击 **Sign in with OpenAI**，并在浏览器中完成 OAuth。
4. 返回 NBG 并选择模型。

说明：
- 不需要填写 API key。
- 可用模型取决于你的 OpenAI plan。
