**Website:** [https://bailian.console.aliyun.com/](https://bailian.console.aliyun.com/)

### 获取 API 凭证

1.  **注册或登录：** 打开 [Alibaba Cloud Bailian Console](https://bailian.console.aliyun.com/)，创建账号或登录现有账号。
2.  **进入 API Keys：** 打开 API key 管理区域。
3.  **创建密钥：** 为应用生成新的 API key。
4.  **复制密钥：** 立即复制 API key，并妥善保存。

### 在 NBG 中配置

> 参考 [认证与模型选择](/getting-started/authorizing-with-cline#menu)。

1.  **打开 NBG 设置：** 在 NBG 面板中点击设置图标 (⚙️)。
2.  **选择 provider：** 在 "API Provider" 下拉菜单中选择 "Alibaba Qwen"。
3.  **选择区域：** 根据账号和使用区域选择 "International" 或 "China"。
4.  **填写 API key：** 将 Qwen API key 粘贴到 "Qwen API Key" 字段。
5.  **选择模型：** 在 "Model" 下拉菜单中选择需要使用的模型。

### 提示

-   **视觉支持：** VL (Vision-Language) 模型支持图像输入，可用于多模态任务。
-   **价格：** 当前价格以 [Alibaba Cloud Bailian Console](https://bailian.console.aliyun.com/) 为准。
