## NBG Shared Configuration

1. 打开 NBG 设置 (⚙️)。
2. 在 **API Provider** 中选择目标 provider。
3. 在对应凭证字段中粘贴 API key、token 或其他凭证。
4. 在 **Model** 中选择模型。

完整认证流程，包括 IDE 和 CLI，请参考 [认证与模型选择](/getting-started/authorizing-with-cline#menu)。

## Providers

### AIHubMix
AIHubMix 是 OpenAI-compatible 模型聚合服务，可通过一个 API surface 访问多个模型后端。

**Website:** [https://aihubmix.com/](https://aihubmix.com/)

#### 基本配置

1. 创建或登录 AIHubMix 账号。
2. 在 dashboard 中生成 API key。
3. 在 NBG 中选择 **AIHubMix** 并粘贴 key。
