作为团队成员，你可以把本地开发环境连接到组织的 Anthropic provider 配置。本指南介绍如何在 VS Code 中配置 API key，从而通过组织配置使用 Claude models。管理员已经完成 provider settings 配置，你只需要添加 API key 即可开始。

## 开始之前

要成功连接组织的 Anthropic provider，需要先准备以下内容。

**已安装并配置 NBG**

需要在 VS Code 中安装 NBG，并登录组织账号。如果尚未安装 NBG，请参考 [安装指南](/getting-started/installing-cline)。

<Info>
**快速检查**：在 VS Code 中打开 NBG panel。如果左下角能看到组织名称，说明已经正确登录。
</Info>

**Anthropic API key**

你需要 Anthropic API key 来认证请求。组织可能会集中提供 keys，也可能要求你通过 [Anthropic Console](https://console.anthropic.com/) 自行创建。

<Note>
如果不确定如何获取 API key，请向管理员确认组织的 key provisioning process。
</Note>

## 配置步骤

<Steps>
<Step title="打开 NBG Settings">
打开 VS Code，并使用以下任一方式进入 NBG settings panel：

- 点击 NBG panel 中的 settings icon
- 点击聊天区域正下方的 API Provider dropdown

</Step>

<Step title="输入 API Key">

1. 选择或确认已选择 **Anthropic** provider
2. 在 **API Key** 字段中输入 Anthropic API key
3. 如果管理员配置了 custom base URL，它会已经设置并锁定
4. 点击 **Save** 保存凭据

<Tip>
API keys 会保存在本地，并且只由 NBG 使用。
</Tip>

<Note>
Base URL setting 由管理员控制。如果配置了 custom proxy URL，API requests 会自动通过它路由。
</Note>
</Step>

<Step title="验证配置">
输入 API key 后，管理员控制的 settings（例如 base URL）会被锁定（显示 lock icon），因为它们由组织统一管理。
</Step>

<Step title="测试连接">
在 NBG 中发送一条测试消息，确认 API key 可以正确访问已配置的 Anthropic endpoint。

<Tip>
**测试建议**

开始开发任务前，先用类似 `Hello` 的简单消息验证基础连接。
</Tip>
</Step>
</Steps>

## Troubleshooting

**Anthropic not available as provider option**

Confirm you're signed into the correct NBG organization. Verify your administrator has saved the Anthropic configuration and that you have the latest version of NBG.

**Authentication errors ("Invalid API Key" or "Unauthorized")**

Verify your API key is correct and active. Check the [Anthropic Console](https://console.anthropic.com/) to confirm your key status and that it has sufficient permissions.

**Connection errors or timeouts**

If your administrator configured a custom base URL (proxy), check with your IT team about network requirements. If using the default Anthropic endpoint, ensure you have internet access to `api.anthropic.com`.

**Models not available**

The available models depend on your Anthropic API plan and your organization's configuration. Contact your administrator if expected models are not available.

**Rate limit errors**

Your API key may have rate limits configured by Anthropic. If you encounter rate limit errors during normal use, contact your administrator about adjusting limits or managing key usage across the team.

## 安全最佳实践

使用 Anthropic API key 时：

- 妥善保管 API key，不要共享
- 不要把 API key 存入代码或版本控制
- 如果怀疑 key 泄露，立即报告管理员
- 定期检查 [Anthropic Console](https://console.anthropic.com/) 是否有异常 usage patterns

更多细节见 [Anthropic API documentation](https://docs.anthropic.com/)，并请与组织管理员协调。
