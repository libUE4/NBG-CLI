NBG Enterprise API 提供账号管理、组织管理、billing 和 API key 管理 endpoints。它与负责模型推理的 [Chat Completions API](/api/overview) 分离。

## Base URL

```
https://<your-nbg-enterprise-api>
```

## Authentication

所有 endpoints 都需要在 `Authorization` header 中携带 Bearer token：

```bash
Authorization: Bearer YOUR_AUTH_TOKEN
```

可使用 [公开 API 文档](/api/overview#authentication) 中描述的 API key 或 account auth token。

## 快速示例

```bash
export NBG_ENTERPRISE_API_BASE_URL="https://<your-nbg-enterprise-api>"
export NBG_AUTH_TOKEN="YOUR_AUTH_TOKEN"

# 获取当前用户资料
curl "$NBG_ENTERPRISE_API_BASE_URL/api/v1/users/me" \
  -H "Authorization: Bearer $NBG_AUTH_TOKEN"
```

```json
{
  "id": "user_abc123",
  "email": "you@company.com",
  "name": "Your Name",
  "active_account_id": "org_xyz789"
}
```

## Users

管理用户账号、terms、balance、usage 和 payment methods。

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/users/me` | 获取当前用户资料 |
| `PATCH` | `/api/v1/users/me` | 更新当前用户资料 |
| `DELETE` | `/api/v1/users/me` | 删除当前用户账号 |
| `POST` | `/api/v1/users/me/accept-terms` | 接受服务条款 |
| `GET` | `/api/v1/users/me/remote-config` | 获取当前用户的 remote configuration |
| `PUT` | `/api/v1/users/active-account` | 切换 active account，个人账号或组织账号 |
| `GET` | `/api/v1/users/{id}/balance` | 获取 credit balance |
| `GET` | `/api/v1/users/{id}/usages` | 获取 usage history |

### Payments and Credits

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/users/{id}/payments` | 列出 payment history |
| `GET` | `/api/v1/users/{id}/payments/{paymentId}` | 获取 payment details |
| `GET` | `/api/v1/users/{id}/payments/{paymentId}/status` | 检查 payment status |
| `GET` | `/api/v1/users/{id}/payments/provider/{paymentId}` | 获取 provider-side payment details |
| `POST` | `/api/v1/users/credits/checkout` | 发起 credit purchase checkout |
| `POST` | `/api/v1/users/{id}/credits/purchase` | 直接购买 credits |

### Billing Configuration

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/users/{id}/auto-top-up` | 获取 auto top-up settings |
| `PUT` | `/api/v1/users/{id}/auto-top-up` | 配置 auto top-up |
| `GET` | `/api/v1/users/{id}/payment-method/default` | 获取 default payment method |
| `POST` | `/api/v1/users/{id}/payment-method/setup-session` | 发起 payment method setup |
| `GET` | `/api/v1/users/{id}/promotions` | 列出 active promotions |

## Organizations

创建并管理 organizations。组织管理员可以配置 remote settings、管理 members 并控制 billing。

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `POST` | `/api/v1/organizations` | 创建 organization |
| `GET` | `/api/v1/organizations/{id}` | 获取 organization details |
| `PUT` | `/api/v1/organizations/{id}` | 更新 organization settings |
| `DELETE` | `/api/v1/organizations/{id}` | 删除 organization |
| `GET` | `/api/v1/organizations/{id}/api-keys` | 列出 organization API keys |
| `GET` | `/api/v1/organizations/{id}/remote-config` | 获取 organization remote config |
| `GET` | `/api/v1/organizations/{orgId}/metrics` | 获取 organization usage metrics |

## Organization Members

管理组织访问成员及其角色。

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/organizations/{orgId}/members` | 列出所有 members |
| `DELETE` | `/api/v1/organizations/{orgId}/members` | 移除 members |
| `GET` | `/api/v1/organizations/{orgId}/members/available-roles` | 列出可分配 roles |
| `PUT` | `/api/v1/organizations/{orgId}/members/{memberId}/role` | 修改 member role |
| `GET` | `/api/v1/organizations/{orgId}/members/{memberId}/usages` | 获取 member usage |

<Tip>
  UI 中的成员管理流程见 [Managing Members](/enterprise-solutions/team-management/managing-members)。
</Tip>

## Organization Invites

邀请新成员加入 organization。

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/organizations/{orgId}/invites` | 列出 pending invites |
| `POST` | `/api/v1/organizations/{orgId}/invites` | 发送新 invites |
| `GET` | `/api/v1/organizations/{orgId}/invites/count` | 获取 invite count |
| `DELETE` | `/api/v1/organizations/{orgId}/invites/{inviteId}` | 撤销 invite |
| `POST` | `/api/v1/invites/accept` | 接受 invite，由 invitee 调用 |

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/organizations/{orgId}/balance` | 获取 org credit balance |
| `GET` | `/api/v1/organizations/{orgId}/payments` | 列出 payment history |
| `GET` | `/api/v1/organizations/{orgId}/payments/{paymentId}` | 获取 payment details |
| `GET` | `/api/v1/organizations/{orgId}/payments/{paymentId}/status` | 检查 payment status |
| `GET` | `/api/v1/organizations/{orgId}/payments/provider/{paymentId}` | 获取 provider-side payment details |
| `POST` | `/api/v1/organizations/{orgId}/credits/checkout` | 发起 credit checkout |
| `POST` | `/api/v1/organizations/{orgId}/credits/purchase` | 购买 credits |
| `GET` | `/api/v1/organizations/{orgId}/auto-top-up` | 获取 auto top-up config |
| `PUT` | `/api/v1/organizations/{orgId}/auto-top-up` | 配置 auto top-up |
| `GET` | `/api/v1/organizations/{orgId}/payment-method/default` | 获取 default payment method |
| `POST` | `/api/v1/organizations/{orgId}/payment-method/setup-session` | 发起 payment method setup |
| `GET` | `/api/v1/organizations/{id}/promotions` | 列出 active promotions |

## Organization Plans

订阅、升级或取消 plans，并管理团队 seats。

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/plans` | 列出可用 plans |
| `GET` | `/api/v1/organizations/{orgId}/plan` | 获取当前 plan |
| `GET` | `/api/v1/organizations/{orgId}/plan/history` | 查看 plan change history |
| `GET` | `/api/v1/organizations/{orgId}/plan/{planId}` | 获取指定 plan details |
| `POST` | `/api/v1/organizations/{orgId}/plan` | 订阅 plan |
| `PUT` | `/api/v1/organizations/{orgId}/plan/seats` | 更新 seat count |
| `DELETE` | `/api/v1/organizations/{orgId}/plan/{planId}` | 取消 plan |

## API Keys

创建并管理用于程序化访问的 API keys。这里创建的 keys 可用于 [Chat Completions API](/api/overview) 和本页 endpoints，具体取决于你的网关权限策略。

| Method | Endpoint | 说明 |
|--------|----------|-------------|
| `GET` | `/api/v1/api-keys` | 列出 API keys |
| `POST` | `/api/v1/api-keys` | 创建 API key |
| `DELETE` | `/api/v1/api-keys/{key_id}` | 删除 API key |
