## 什么是 NBG API？

NBG API 是 OpenAI-compatible Chat Completions endpoint。你可以把 Anthropic、OpenAI、Google、自托管模型或其他兼容 provider 接到同一个 base URL 后面，应用侧只需要使用统一 API key 和模型 ID。

```
Your App  ->  NBG API Gateway  ->  Anthropic / OpenAI / Google / local models / etc.
```

## 快速开始

```bash
export NBG_API_BASE_URL="https://<your-nbg-api>/api/v1"
export NBG_API_KEY="YOUR_API_KEY"

curl -X POST "$NBG_API_BASE_URL/chat/completions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "anthropic/claude-sonnet-4-6",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

API key 应由你的 NBG API gateway 或企业控制台签发。旧 Cline 托管 dashboard 不作为 NBG 默认商业化入口。
